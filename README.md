# test-api-express
> Full CRUD REST API Built with Express

This is a REST API  built for learning purposes, it is set up to expose endpoints that are useful for my projects.

```
# Available Routes
GET      /api/products
POST     /api/products
GET      /api/products/:id
PUT      /api/products/:id
DELETE   /api/products/:id

```

## Usage

```
# Install dependencies
npm install
yarn install

# Run in develpment
npm run dev
yarn run dev

# Run in production
npm start
yarn start
```
